LiveASM
=======

A simple machine code emitting library intended for use as a JIT or code generation backend written in C++.

Architectures:
==============

In-progress:
------------
* x86_64 / AMD64

Planned:
--------
* x86_64.MMX
* x86_64.SSE2
* x86_64.SSE3
* x86_64.SSE_4_1
* x86_64.SSE_4_2
* i686
* i686.MMX
* i686.SSE2
